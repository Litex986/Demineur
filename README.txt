======== Fonctionnement =========

Afin de compiler le programme lancer le fichier makefile à l'aide de la commande "make".

Pour naviguer dans les menus utiliser les indices avant l'action voulut.
Pour choisir une case entrer les coordonnées séparés par une virgule.

Bonne chance.
